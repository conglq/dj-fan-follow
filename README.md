# dj-fan-follow
SPOON CODING TEST


## Enviroment requirement

### NodeJS v12.x
### Redis service v4.x


## Install

### Install dependency 

```sh
npm install
```

## Run

### Config
Can edit configuration **[http port, redis connection url]** by process env or ./config/index.js file

### Start service

```sh
npm start
```

### Run Tests

```sh
npm test
```
